## 概要

画像のピクセルやオセロの盤面のような構造を一次元配列で管理しているが、インデックスを(x, y)に変換したい、もしくはその逆の計算をする場合のアルゴリズムです。

## コード例

### インデックスを一次元から二次元に変換する場合

```typescript
const x = index % width;
const y = Math.floor(index / width);
```

### インデックスを二次元から一次元に変換する場合

```typescript
const index = y * width + x;
```
