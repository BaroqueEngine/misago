## 概要

![](./static/images/3385ebb2/0.png)

`マンハッタン距離`は距離の計算方法の一つで、軸ごとに[差の絶対値の距離](/b98d6da4)を測り、その距離の総和を求めます。

## コード例

```typescript
function manhattanDistance(
  x1: number,
  y1: number,
  x2: number,
  y2: number
): number {
  return Math.abs(x1 - x2) + Math.abs(y1 - y2);
}
```
