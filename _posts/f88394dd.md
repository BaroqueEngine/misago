## 概要

![](./static/images/f88394dd/0.png)

チェス盤距離は、軸ごとに[差の絶対値の距離](/b98d6da4)を測り、一番長い距離を返します。

## コード例

```typescript
function chessboardDistance(
  x1: number,
  y1: number,
  x2: number,
  y2: number
): number {
  const x = Math.abs(x1 - x2);
  const y = Math.abs(y1 - y2);
  return Math.max(x, y);
}
```
