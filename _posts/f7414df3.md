## 概要

渡された値が 0 なら 0、0 未満なら-1、1 以上なら 1 を返します。

## コード例

```typescript
function dir(value: number): number {
  if (value === 0) {
    return 0;
  } else if (value < 0) {
    return -1;
  } else {
    return 1;
  }
}
```

```typescript
console.log(dir(-2)); // -1
console.log(dir(-1)); // -1
console.log(dir(0)); // 0
console.log(dir(1)); // 1
console.log(dir(2)); // 1
```
