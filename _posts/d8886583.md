## 概要

数直線上で、ある実数 a が原点からどれだけ離れているか、つまり原点から実数 a までの距離を`aの絶対値`といい、 $|a|$ と表す。

- $|2| = 2$
- $|-2| = 2$
- $|\sqrt{2}| = \sqrt{2}$
- $|-\sqrt{2}| = \sqrt{2}$

絶対値はもとの値の符号が取れた形になるので、一般に $|x|=|-x|$ が成り立つ。

## コード例

```typescript
Math.abs(5); // 5
Math.abs(-10); // 10

Math.abs(2) === Math.abs(-2); // true
```
