## 解説/アルゴリズム

2 つの円 $C_1, C_2$ があり、それぞれの半径を $r_1, r_2\thinspace(r_1 > r_2)$ 、中心間の距離を $d$ とする。

このとき、2 つの円の位置関係は下記のとおりとなる。

| 条件                    | 結果                               |
| :---------------------- | :--------------------------------- |
| $d > r_1+r_2$           | 離れている                         |
| $d = r_1+r_2$           | 互いに外接している                 |
| $r_1-r_2 < d < r_1+r_2$ | 2 点で交わる                       |
| $d = r_1-r_2$           | 円 $C_2$ が円 $C_1$ に内接している |
| $d < r_1-r_2$           | 円 $C_2$ が円 $C_1$ の内部にある   |
