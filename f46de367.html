平均

解説/アルゴリズム

x1,x2...,xnx_1,x_2...,x_nx1​,x2​...,xn​ と n 個のデータがあるとき、データの総和を大きさ n で割ることを平均と呼びます。

平均で求まる値のことを平均値と呼びます。

μ\muμを平均値とする場合、μ=x1+x2+...+xnn\mu = \dfrac{x_1+x_2+...+x_n}{n}μ=nx1​+x2​+...+xn​​ となります。

// データ
const data = [88, 51, 40, 16, 76, 2, 12, 84];
// データの総和
const total = data.reduce((a, b) => a + b, 0);
// 平均値を求める
const avg = total / data.length;
console.log(avg); // 46.125

度数分布表の平均

階級値 度数
x1x_1x1​ f1f_1f1​
x2x_2x2​ f2f_2f2​
... ...
xnx_nxn​ fnf_nfn​

度数分布表の平均を取る場合は、各階級のデータがすべてその階級値に等しいと考えて、各階級の階級値に度数を掛けた値の総和を階級の数で割ります。

μ=(x1f1+x2f2+...+xnfn)n\mu=\dfrac{(x_1f_1+x_2f_2+...+x_nf_n)}{n}μ=n(x1​f1​+x2​f2​+...+xn​fn​)​

type Fd = {
  level: number;
  count: number;
};

const data: Fd[] = [
  { level: 0, count: 0 },
  { level: 10, count: 13 },
  { level: 20, count: 28 },
  { level: 30, count: 10 },
  { level: 40, count: 18 },
];
const total = data.reduce((a, b) => a + b.level * b.count, 0);
const avg = total / data.length;

console.log(avg); // 342

変形

μ=x1+x2+...+xnn\mu = \dfrac{x_1+x_2+...+x_n}{n}μ=nx1​+x2​+...+xn​​

の両辺に n を掛けて、

nμ=x1+x2+...xnn \mu = x_1+x_2+...x_nnμ=x1​+x2​+...xn​

この形に変形して利用する場合もあります。
（利用例: 分散）
