チェス盤距離

概要

チェス盤距離は、軸ごとに差の絶対値の距離を測り、一番長い距離を返す。

コード例

function chessboardDistance(
  x1: number,
  y1: number,
  x2: number,
  y2: number
): number {
  const x = Math.abs(x1 - x2);
  const y = Math.abs(y1 - y2);
  return Math.max(x, y);
}

内部で利用しているアルゴリズム

最大値・最小値, 絶対値, 2 点間の直線距離
